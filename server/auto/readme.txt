#---Setup auto farm---#
1. auto.c 		=> copy to current/cmd/base/
2. runauto.c	=> copy to current/cmd/user/
3. tool.c		=> copy to current/item/<folder>/
4. title.c		=> copy to current/sys/sys/

#---Hướng dẫn---#
1. auto.c		=> file xử lý lựa chọn chức năng từ giao diện game
2. runauto.c	=> file chức năng
3. tool.c		=> vật phẩm bật/tắt auto
4. title.c		=> danh hiệu khi chạy auto (thay đổi dòng 186 và dòng 374->381)
